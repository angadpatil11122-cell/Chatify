🚀 Chatify Product Roadmap

This roadmap is a high-level overview of the features and updates we are currently working on or planning for Chatify. Note that all dates and features are subject to change based on development progress and security reviews.

Q4 2025: Current Focus (In Progress)

In Progress

Multi-Device Login and Sync

Allow users to seamlessly link and sync their Chatify account across multiple devices (Desktop, Web, Mobile) without compromising E2EE key security.

In Progress

Encrypted Voice and Video Calls

Implement E2EE for 1:1 voice and video communication, using secure WebRTC protocols to ensure privacy.

In Progress

Enhanced Profile Customization

Adding options for custom themes, chat bubble colors, and detailed "About Me" sections for user profiles.

Q1 2026: Planned Features

Planned

Group Chat E2EE

Extend End-to-End Encryption support to group chats, ensuring robust key management for multiple participants.

Planned

Desktop Application (Native)

Launch dedicated native apps for Windows and macOS for a smoother, more integrated desktop experience.

Recently Completed (Q3 2025)

Completed

End-to-End Encrypted File Transfer

Securely transfer photos and documents up to 100MB using the E2EE protocol.

Completed

Username-Based Search

Implemented fast, unique username search to easily connect with friends.
